Hackathon Prototype

Volunteer Skill-Matching Engine

Two-role platform for skill-first volunteer matching.

Volunteers create their own profile and availability. Organisers log in separately, post a requirement, and the system ranks only the volunteers who actually signed up.

I am a volunteer I am an organiser
Role-based separate entry pages
Live data from volunteer signup
Smart match skill, time, distance, trust

Prototype flow

Register volunteers first, then match them from the organiser dashboard.

  • Volunteer sign up and login
  • Organiser sign up and login
  • Volunteer profile stored in browser
  • Organiser sees only registered volunteers

Demo tip

3 steps

Create 2-3 volunteer profiles with different skills, then log in as an organiser and show the ranking engine in action.

How to demo

Simple and judge-friendly flow

01

Volunteer fills profile

Name, skill, address, city, trust score, and available time slots are stored for matching.

02

Organiser posts need

The organiser chooses the exact skill, address, city, urgency, start time, and duration.

03

Engine ranks matches

The system recommends the strongest volunteer using only the profiles created in the app.
